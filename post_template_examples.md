# Post Template Examples

These examples can be copied into the `post_template` field in the matching JSON config file.

## `post_config_mahakali.json`

```json
"post_template": "Today's name is {name}. {meaning} Jai Ma. #Mahakali"
```

```json
"post_template": "{name}: {meaning} Jai Ma. #Mahakali #Devi"
```

```json
"post_template": "Meditating on {name}: {meaning} Jai Ma. #Mahakali"
```

```json
"post_template": "Name of the day: {name}. {meaning} Jai Ma. #Shakti"
```
